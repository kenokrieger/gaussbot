rendering.save_as_png(expr, filename, is_latex=False, dpi=300)¶

Saves a sympy expression as a png with the help of matplotlib.

Parameters

  • expr (sympy.core or latex str) – A sympy expression that shall be displayed.

  • filename (str or path) – The path where the figure will be saved to.

  • is_latex (bool) – Specifies whether the string is already in latex format or needs to be parsed.

  • dpi (int) – The resolution of the image in dots per inch
